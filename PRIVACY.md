# Politique de confidentialité — Quiet Web

**Dernière mise à jour :** 9 mai 2026  

**Quiet Web** est une extension de navigateur publiée par **3Devs**. Cette politique décrit les données traitées **par l’extension sur votre appareil**.

## Données collectées par l’éditeur (3Devs)

**Aucune.** Quiet Web ne contacte pas de serveur exploité par **3Devs** pour collecter des données personnelles, ne contient pas de télémétrie, ni de publicité, ni de suivi tiers.

## Données stockées localement

L’extension utilise le stockage local du navigateur (`storage`), uniquement sur votre machine, pour :

- vos **réglages** (activation, mode, exclusions de sites, règles personnalisées) ;
- des **statistiques agrégées** (nombre d’éléments masqués par le script de page, ventilées par type de détection).

Ces informations peuvent être **exportées** par vous au format JSON depuis la page d’options. Elles ne sont **pas** transmises automatiquement à un serveur.

## Permissions du navigateur

- **storage** : enregistrer la configuration et les statistiques localement.
- **tabs** : connaître l’onglet actif pour les exclusions par site, la réanalyse et les raccourcis.
- **Accès aux pages (`<all_urls>`)** : exécuter des scripts de contenu pour masquer des éléments intrusifs sur les sites que vous visitez.

## Sites web visités

Le traitement a lieu **dans votre navigateur** lors de la visite de pages. L’extension ne transfère pas la liste de vos URL vers un backend Quiet Web.

## Contact

Pour toute question relative à cette politique, contactez **3Devs** via la page du magasin d’extensions (Opera Add-ons, Chrome Web Store, etc.) ou le canal de support indiqué sur la fiche de **Quiet Web**.

## Modifications

**3Devs** peut mettre à jour cette politique. La version applicable est celle publiée avec l’extension ou sur l’URL de politique indiquée dans la fiche du magasin.
