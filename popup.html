Quiet Web

Réduit les bannières cookies et les modales intrusives. Raccourcis : Alt+Shift+Q activer/désactiver, Alt+Shift+R réanalyser.

Masquer une bannière ne remplace pas un choix de consentement (cookies, données personnelles). À utiliser en connaissance de cause.

Statistiques (local)
Total masqué0
Via sélecteurs0
Via analyse (Max)0

Compte chaque élément masqué par le script de page. L’injection très tôt (CSS) n’est pas comptée.

—

Exclure : Quiet Web ne masque rien sur ce domaine (l’extension peut rester activée ailleurs).

Prudent : sélecteurs connus + les vôtres uniquement (pas d’analyse texte des overlays). Max : comportement complet.

Cookies : CMP + textes consentement. Modales : anti-adblock, newsletters, paywall, etc. Tout : les deux.

Ex. #my-banner ou .overlay--newsletter. Lignes vides ou commençant par // ignorées.

Utilisé seulement en mode Max. Si l’overlay contient cette chaîne (sans casse), masquage possible avec garde-fous.
